# Business Sales Performance Analytics
## Overview
This project focuses on analyzing customer churn and retention patterns using a real-world subscription dataset. The goal is to identify why customers leave, which segments are most at risk of churn, and how customer lifetime behavior impacts retention.
The analysis simulates a real SaaS or subscription-based business scenario and provides actionable business recommendations to improve customer retention.
## Dataset
Dataset: Telco Customer Churn Dataset (Kaggle)
Description: Contains customer demographics, subscription details, tenure, service usage, and churn status.
## Tools Used
Power BI – Data cleaning, visualization, and dashboard creation
Excel – Basic data validation
## Key Analysis Performed
Churn rate and retention metrics
Churn analysis by contract type, tenure, and services
Customer lifetime (tenure-based) analysis
Interactive dashboard with filters
## Key Insights
Month-to-month customers have the highest churn rate
Customers churn most within the first 6 months
Long-tenure customers demonstrate strong retention
Higher monthly charges are associated with increased churn
## Business Recommendations
Promote long-term contracts
Improve early customer onboarding
Enhance value proposition for high-paying customers
## Conclusion
This project demonstrates how data analytics can directly support business decision-making by identifying churn drivers and proposing data-driven retention strategies.
